# copilot-organization-deno

## 概要
DenoとCopilot CLIを組み合わせて、タスク自動化を行う実験用リポジトリ。

## 主なファイル
- `ai-team.ts` : 複数エージェントをDenoで管理し、タスク分担・実行を行うサンプル。

## 使い方
1. Copilot CLIがインストールされていることを確認してください。
2. 必要に応じて `deno.json` のパスやバージョンを調整してください。
3. サンプル実行例:
   ```sh
   deno run -A ai-team.ts '昨日の日経平均を簡単にまとめて'
   ```

## 注意事項
- Copilot CLIのバージョンやエージェント仕様により動作が変わる場合があります。
- ポート競合や権限エラー時は、エラーメッセージを参考に設定を見直してください。
